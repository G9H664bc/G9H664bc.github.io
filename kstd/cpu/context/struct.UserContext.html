pub fn activate_tls_pointer(&self)

Activates thread-local storage pointer on the current CPU.

§Safety

The method by itself is safe because the value of the TLS register won’t affect kernel code. But if the user relies on the TLS pointer, make sure that the pointer is correctly set when entering the user space.
